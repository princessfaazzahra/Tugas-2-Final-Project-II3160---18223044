## Tentang Aplikasi: Platoo

Platoo hadir sebagai platform digital yang dirancang untuk menjawab permasalahan food waste melalui distribusi dan pemanfaatan makanan tidak terjual. Platform ini menghubungkan restoran dengan konsumen dalam satu ekosistem terintegrasi.  

Pihak restoran dapat mempublikasikan makanan yang masih layak konsumsi namun berisiko tidak terjual, seperti produk mendekati masa kedaluwarsa atau stok berlebih. Setiap produk ditampilkan dalam katalog dengan informasi yang jelas dan terstruktur, mencakup nama, foto, harga, kualitas, jumlah, serta lokasi dan waktu pengambilan.  

Pendekatan berbasis data ini memungkinkan proses distribusi dilakukan secara lebih efisien dan transparan.

Untuk pemenuhan Tugas 2, telah ditentukan microservice yang akan dijadikan fokus pengembangan dari aplikasi Platoo yaitu **Layanan Katalog Makanan**.  

Pada microservice ini terdapat 3 API utama yang nantinya akan diintegrasikan pada Tugas 3. Ketiga API tersebut memiliki tugas untuk:
1. Mengambil seluruh katalog makanan  
2. Mengubah status ketersediaan makanan  
3. Mengedit makanan, seperti mengubah harga, nama makanan, dan stok  

Hasil dari microservice tersebut telah diisolasi menggunakan Docker dan di-*deploy* pada STB sehingga API dari microservice **Layanan Katalog Makanan** dapat diakses oleh publik.
